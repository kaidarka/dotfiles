# Настройка

Для применения конфигурации файл .tmux.conf нужно скопировать в домашнюю дирректорию.

## Дополнения

Для работы плагинов, указанных в конфигурации требуется установка менеджера
https://github.com/tmux-plugins/tpm

Для установки плагинов требуется при следующем запуске нажать `<prefix>+I`
